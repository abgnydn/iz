iz-1 v0 is a ternary multi-modal Earth-observation regression model (≈100M params → ~20 MB on-disk) for industrial CO₂ verification. Browser inference uses fused WGSL kernels derived from fused-lora for the LoRA adapter delta, layered on top of Transformers.js / ONNX Runtime WebGPU for the base ViT.
